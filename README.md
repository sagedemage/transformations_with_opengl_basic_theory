# Transformations with OpenGL - Basic Theory

Transformations with OpenGL basic theory article.

## Setup

Install all dependencies
```
pnpm install
```

Start the website
```
pnpm docs:dev
```

Build the website
```
pnpm docs:build
```

## Why use pnpm?
1. Save disk space
2. Boost installation speed
